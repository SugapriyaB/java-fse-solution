Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

There are several ways to install Bootstrap:
  • Include via CDN
  • Install via package manager (npm)
  • Download source files
  • Include in your project as a module

Yes, Bootstrap is mobile-first! Since version 3, Bootstrap has been mobile-first by design. It uses a responsive grid system that allows your website to automatically adjust its layout based on the screen size.

Yes, Bootstrap is highly customizable. You can:
  • Override default variables
  • Create custom themes
  • Modify the Sass files
  • Use Bootstrap's built-in customize tools
